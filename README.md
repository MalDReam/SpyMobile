# SpyMobile

Перед вами мобильная игра "Шпион", предназначенная для компании от 3-ех человек. Веселитесь! 

## Как играть?

**1.** Ознакомьтесь с правилами
**2.** Настройте игру под компанию
**3.** Раздайте роли
**4.** Играйте! Если необходимо - пользуйтесь подсказками

## Предупреждение
Мой слабенький ноут не тянет выгрузку в APK, поэтому тут чисто код. Если ноут не помрет, то в ближайшее время выложу и APK.
